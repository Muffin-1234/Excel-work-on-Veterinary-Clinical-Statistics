# Excel-work-on-Veterinary-Clinical-Statistics
This project is to display the condition of various breeds and species of animals which were brought to a certain clinic at a particular time. The goal is to clean, explore and derive meaningful insights from the data using Excel.

## Dataset description
The dataset used in this project contains information about animals brought in a veterinary clinic.

Key fields include:
- Column 1: Animal types
- Column 2: Animal breeds
- Column 3: Animal age
- Column 4: Animal gender
- Column 5: Animal weight
- Column 6: Prominent symptom
- Column 7: Symptom duration in days
- Column 8: Body temperature
- Column 9: Heart rate
- Column 10: Disease prediction

## Tools used
- Microsoft Excel
- Pivot Tables
- Excel formulas
- Charts
